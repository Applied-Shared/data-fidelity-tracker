Regenerating a report

Run inside the dev docker in core-stack, then copy the output into this repo and redeploy. 

bazel run //tools/offboard/common/image_quality_analysis:neural_sim_report -- \
  --lance_index_path s3://onroad-perception-datasets/scenario_dataset_export/gen2_mache_scenario_dataset/2026-04-29_17-51-19/dataset.lance \
  --title "Baseline gen2_mache_scenario_dataset data quality report" \
  --dt_min 2025-07-01 --dt_max <today> \
  --summary_by segment

Then copy the output report.html → static/sections/gen2_mache_baseline/index.html and the images/ dir alongside it, commit, and run apps-platform app deploy.
